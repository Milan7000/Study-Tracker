# 📚 Study Tracker Web App

A clean, minimal, and interactive **Study Tracker** designed to help students systematically track syllabus completion for competitive exams like **GATE (ECE)** and **SBI PO**.

This project focuses on clarity, usability, and real exam preparation workflows—turning a simple to-do concept into a structured learning companion.

---

## 🚀 Features

### 🎯 Core Functionality
- Select exam:
  - GATE (Electronics & Communication)
  - SBI PO
- Structured syllabus:
  - Subject → Topics hierarchy
- Mark topics as:
  - ✅ Completed
  - ⬜ Pending
- Automatic **progress calculation (%)**
- Visual **progress bar**

---

### ⭐ Smart Tracking
- Topic importance indicators (priority-based ⭐ system)
- Real-time updates on:
  - Completed topics
  - Remaining syllabus

---

### 🎨 UI/UX
- Clean and minimal interface
- Smooth and responsive design
- Light/Dark mode toggle 🌙
- Card-based layout for better readability

---

### 💾 Data Persistence
- Uses **localStorage**
- Your progress is محفوظ (saved) even after refresh

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling (modern UI + dark mode)  
- **Vanilla JavaScript** – Logic & interactivity  

No frameworks. No libraries. Pure frontend fundamentals.

## ⚙️ How It Works

1. Select an exam from the dropdown
2. Browse subjects and topics
3. Mark topics as completed
4. Progress updates automatically
5. Toggle dark mode if needed
6. Your data is saved locally in the browser

---

## 🧠 Design Philosophy

This project is built around a simple idea:

> “What gets measured gets completed.”

Instead of overwhelming dashboards, the app focuses on:
- Clarity over complexity  
- Action over decoration  
- Progress over perfection
